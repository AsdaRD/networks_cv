Ecosmart

Программно-аппаратный комплекс диспетчерского управления автоматизированной системой экологического мониторинга. Построено на стеке технологий PHP (Yii2), MYSQL, Python.

Функции комплекса:

  • сбор, обработка и отображение данных датчиков;
  • синхронизация данных с сервером;
  • создание резервных копий;
  • восстановление настроек контроллера с сервера;
  • система событий;
  • гибкое управление правами пользователей.

EcoMap

Система мониторинга состояния внешней природной среды. Построено на стеке технологий NodeJS, Postgres, VueJS.

Система позволяет:

  • непрерывно получать, обрабатывать и передавать данные от автоматических станций контроля качества физико-химических параметров окружающей среды;
  • отображать данные, введенные вручную;
  • отображать различные типы объектов, влияющих на состояние окружающей среды;
  • создавать резервные копии;
  • управлять правами пользователей.

IncognitoHR

Глобальная рекрутинговая платформа для IT-специалистов, обеспечивающая безопасный и анонимный набор персонала. Работает на стеке PHP, MYSQL.

Нашей командой было реализовано:

  • автоматические бэкапы;
  • процесс разработки по методологии KANBAN;
  • большое количество frontend и backend задач.

MikroTik backup and monitoring

Система, позволяющая мониторить состояние VPN и создавать резервные копии роутеров компании MikroTik. Построено на стеке технологий VueJS, NodeJS, Strapi.

Функции системы:

  • выпуск серверов за роутером фирмы MikroTik через IkeV2 VPN;
  • мониторинг состояния VPN;
  • создание резервных копий роутеров фирмы MikroTik на Google drive;
  • уведомление в Telegram при изменении статуса подключения.

Environment connections manager

Менеджер среды подключений. Построено на стеке технологий С# (.Net, WF), NodeJS, Strapi.

Функционал приложения:

  • хранение вашей базы всех возможных подключений к RDP и VPN под вашим личным аккаунтом;
  • получение доступа из любой точки мира при наличии сети Интернет;
  • работа в защищенном формате без добавление истории подключение.

Alarm Button

Система управления серверами по звонку. Построено на стеке технологий PHP, TypeScript, MongoDB.

Функционал приложения:

  • создание сценариев, выполняющихся при звонке;
  • проверка состояния службы прослушки;
  • выполнение сценариев по звонку.
