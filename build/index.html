Наші проекти

Ecosmart

Програмно-апаратний комплекс диспетчерського керування автоматизованою системою екологічного моніторингу. Побудовано на стеку технологій PHP(Yii2), MYSQL, Python.

Функції комплексу:

  • збір, обробка і відображення даних датчиків;
  • синхронізація даних з сервером;
  • створення резервних копій;
  • відновлення налаштувань контролера з сервера;
  • система подій;
  • гнучке управління правами користувачів.
Переглянути

EcoMap

Система моніторингу стану навколишнього природного середовища. Побудовано на стеку технологій NodeJS, Postgres, VueJS.

Cистема дозволяє:

  • безперервно отримувати, обробляти та транслювати дані від автоматичних станцій контролю якості фізико-хімічних параметрів довкілля;
  • відображати дані, введені вручну;
  • відображати різні типи об’єктів, що впливають на стан довкілля;
  • створювати резервні копії;
  • управляти правами користувачів.
Переглянути

IncognitoHR

Глобальна рекрутингова платформа для ІТ-спеціалістів, що забезпечує безпечний та анонімний набір персоналу. Побудовано на стеку PHP, MYSQL.

Нашою командою було реалізовано:

  • технологію безперервної інтеграції та безперервної поставки (CI/CD) за допомогою Jenkins та Docker;
  • кешування даних для підвищення швидкості роботи системи;
  • автоматичні бекапи;
  • процес розробки за методологією KANBAN;
  • велика кількість frontend та backend задач.
Переглянути

MikroTik backup and monitoring

Система, що дозволяє моніторити стан VPN і створювати резервні копії роутерів компанії MikroTik. Побудовано на стеку технологій VueJS, NodeJS, Strapi.

Функції системи:

  • випуск серверів за роутером фірми MikroTik через IkeV2 VPN;
  • моніторинг стану VPN;
  • створення резервних копій роутерів фірми MikroTik на Google drive;
  • повідомлення в Telegram при зміні статусу підключення.

Environment connections manager

Менеджер середовища підключень. Побудовано на стеку технологій С# (.Net, WF), NodeJS, Strapi.

Функціонал дададтка:

  • зберігання вашої бази всіх можливих підключень до RDP і VPN під вашим особистим обліковим записом;
  • отримання достуу з будь-якої точки світу за наявності мережі Інтернет;
  • Робота в захищеному форматі без додавання історії підключення.

Alarm Button

Система управління серверами по дзвінку. Побудовано на стеку технологій PHP, TypeScript, MongoDB.

Функціонал системи:

  • створення сценаріїв, що виконуються при дзвінку;
  • перевірка стану служби прослушки;
  • виконання сценаріїв по дзвінку.
